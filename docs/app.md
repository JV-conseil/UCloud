## Usage

1. Once the Ubuntu Terminal job has started, click on the <kbd>Open interface</kbd> blue button in the top right corner.

2. Navigate to the ucloud folder `cd /work/{your-working-folder}/ucloud`.

3. Then launch the script with `. main.sh`.

4. Now your skeleton should look like this üëá

    ```bash
    /work/{your-working-folder}/
    ‚îú‚îÄ‚îÄ data/
    ‚îú‚îÄ‚îÄ database/
    ‚îú‚îÄ‚îÄ env/
    ‚îú‚îÄ‚îÄ install/
    ‚îú‚îÄ‚îÄ jobs/
    ‚îî‚îÄ‚îÄ ucloud/
        ‚îî‚îÄ‚îÄ main.sh
    ‚îî‚îÄ‚îÄ {your-gh-repo}/
        ‚îî‚îÄ‚îÄ main.py
    ```

5. In case you app process data, do not forget to load files in the `data` folder.

6. Then follow instructions starting after _Do you want to run your Python app_ question.

```warning
Your cloned GitHub repository must have a **`main.py`** file at its root level to be identified as a valid Python üêç app.
```
